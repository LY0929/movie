# MoviesAPP
电影微信小程序APP

使用微信独有的一套标签编写小程序页面。

通过豆瓣公开API接口，获取电影复杂数据。

在小程序中，把代码封装成组件，提高代码的高效运行。

通过大量的JS逻辑编写，实现复杂功能。

小程序项目的完成，使我对接口数据的操作更加娴熟。

通过大量JS逻辑运算，让我对复杂程序编写积累宝贵经验。
